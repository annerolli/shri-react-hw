## Описание

В репозитории находится две части приложения: клиентская и серверная. Для разработки нужно:

1. Запустить клиентск

```
cd client
yarn dev
```

2. Запустить сервер

```
cd simple_api
node server.js
```

3. Открыть `http://localhost:3000`
